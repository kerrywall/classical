a classical music generator

Clicking the button below will generate a random combination of piece type, solo instrument (if applicable) and key, and a YouTube search query for said combination.

This is random, so not all keys are necessarily suited to all instruments.

The YouTube results also won't always produce an exact match, but it should be a fun way to discover classical works.

(Failing that, you can always use the output for composition brainstorming.)
